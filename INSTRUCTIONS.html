AI Newsletter Alternative (HCP Demo)

Quick start instructions for running the local prototype

📋 One-time Setup: Install Node.js

The demo requires Node.js to run on your computer. If you do not have it installed, please complete this step first:

  1. Open in your browser: nodejs.org/en/download
  2. Download the recommended version (Left button: LTS for Long Term Support).
  3. Run the installer (you can accept all default options).
  4. Important: If you already had a terminal window open, close it and open a new one after installation so that Node.js is recognized.

🚀 Step-by-Step Setup Guide

1
Unzip the Project Folder

Windows Right-click the file ai-newsletter-alternative-hcps-new-version-2026-06-02.zip and select "Extract All...". Choose a destination folder (e.g., your Desktop). (Warning: Do not just double-click to look inside the ZIP. You must extract it first for the scripts to run correctly.)

macOS / Mac Double-click the ZIP file. A folder named ai-newsletter-alternative-hcps will automatically appear in the same directory.

2
Open Terminal / Command Prompt

Windows Press the Windows key on your keyboard, type cmd, and press Enter. A black command window will open.

macOS / Mac Hold cmd + Spacebar, type Terminal, and press Enter.

3
Navigate into the Project Folder (Drag & Drop)
  1. In your terminal window, type cd followed by a space (do not press Enter yet).
  2. Drag the extracted folder ai-newsletter-alternative-hcps from your file browser (Finder/Explorer) and drop it directly into the terminal window. The path will fill in automatically.
  3. Press Enter.
4
Install Dependencies

Type the following command in the terminal and press Enter:

npm install

This downloads the required software modules. It takes about 10–30 seconds. You can safely ignore any warning messages during this process.

5
Start the Demo

Type the following command and press Enter:

npm run dev

This starts a local web server. You will see a success message:
Relevance Engine demo running at http://127.0.0.1:5173

Keep this terminal window open while using the demo! Closing the window will stop the app.

6
Open the App in Your Browser

Click the link in your terminal or type the following address into your web browser (e.g., Chrome, Safari, Edge):

👉 http://127.0.0.1:5173/

⚠️ Troubleshooting & Key Notes
  • API Keys: The project includes a .env.local file containing a demo OpenAI API key so that live AI relevance reasoning works out-of-the-box.
    Security Note: This is a private testing package. Do not upload this folder or its files to public repositories.
  • Error: "npm: command not found": Node.js is not installed, or you need to restart your Terminal app to apply the installation path. Close the terminal, reinstall Node.js, and open a new terminal window.
  • Error: "Port 5173 already in use": Another process is occupying the local address. Close all active terminal windows and try running npm run dev again.
  • Stop the Server: To close the demo completely, go to your terminal window, press Ctrl + C (Windows/Mac) or simply close the terminal window.
  • Demo Data only: This app uses simulated patient cases and does not contain or process any real patient health records.
