AI-Newsletter-Alternative (HCP Demo)

Quick-Start-Anleitung zum Ausführen des lokalen Prototyps

📋 Einmalige Voraussetzung: Node.js installieren

Die Demo benötigt Node.js auf Ihrem Computer. Wenn Sie dieses noch nicht installiert haben, führen Sie bitte diesen Schritt zuerst aus:

  1. Öffnen Sie im Browser: nodejs.org/en/download
  2. Laden Sie die empfohlene Version herunter (Schaltfläche links: LTS für Long Term Support).
  3. Führen Sie das Installationsprogramm aus (Sie können alle Standardeinstellungen einfach durchklicken).
  4. Wichtig: Falls Sie bereits ein Terminal geöffnet haben, schließen Sie dieses und öffnen Sie nach der Installation ein neues, damit Node.js erkannt wird.

🚀 Schritt-für-Schritt-Anleitung

1
Projektordner entpacken

Windows Klicken Sie mit der rechten Maustaste auf ai-newsletter-alternative-hcps-new-version-2026-06-02.zip und wählen Sie "Alle extrahieren...". Wählen Sie z. B. Ihren Desktop als Ziel aus. (Achtung: Nicht einfach nur doppelklicken und den Ordner in der ZIP-Ansicht öffnen!)

macOS / Mac Doppelklicken Sie auf die ZIP-Datei. Ein Ordner namens ai-newsletter-alternative-hcps erscheint automatisch.

2
Terminal / Eingabeaufforderung öffnen

Windows Drücken Sie die Windows-Taste auf der Tastatur, geben Sie cmd ein und drücken Sie Enter. Ein schwarzes Fenster (die Eingabeaufforderung) öffnet sich.

macOS / Mac Drücken Sie cmd + Leertaste, geben Sie Terminal ein und drücken Sie Enter.

3
In den Demo-Ordner navigieren (Drag & Drop)
  1. Tippen Sie im Terminal-Fenster den Befehl cd gefolgt von einem Leerzeichen ein (noch nicht Enter drücken!).
  2. Ziehen Sie den entpackten Ordner ai-newsletter-alternative-hcps mit der Maus aus Ihrem Datei-Explorer (Finder/Windows-Explorer) und lassen Sie ihn im Terminal-Fenster fallen. Der Pfad wird automatisch eingefügt.
  3. Drücken Sie nun Enter.
4
Abhängigkeiten installieren

Geben Sie den folgenden Befehl im Terminal ein und drücken Sie Enter:

npm install

Dies lädt die benötigten Module herunter. Das dauert ca. 10 bis 30 Sekunden. Eventuelle Warnungen im Terminal können Sie einfach ignorieren.

5
Demo starten

Geben Sie den folgenden Befehl ein und drücken Sie Enter:

npm run dev

Es wird ein lokaler Webserver gestartet. Sie sehen die Erfolgsmeldung:
Relevance Engine demo running at http://127.0.0.1:5173

Lassen Sie dieses Terminal-Fenster während der Benutzung unbedingt geöffnet! Wenn Sie es schließen, beenden Sie die App.

6
App im Browser öffnen

Klicken Sie im Terminal auf den angezeigten Link oder tippen Sie folgende Adresse in Ihrem Browser (z. B. Chrome, Safari, Edge) ein:

👉 http://127.0.0.1:5173/

⚠️ Wichtige Hinweise
  • API-Schlüssel: In der ZIP-Datei befindet sich bereits die Datei .env.local mit einem voreingestellten OpenAI-Demo-Schlüssel. Der Test funktioniert sofort live, ohne dass Sie etwas konfigurieren müssen. Geben Sie die Datei aus Sicherheitsgründen bitte nicht öffentlich weiter.
  • Nur Testdaten: Das System läuft rein mit synthetischen, fiktiven Patientenfällen. Es werden zu keinem Zeitpunkt echte Patientendaten verarbeitet.
  • Demo beenden: Um den Server zu beenden, gehen Sie zurück ins Terminal-Fenster und drücken Sie die Tastenkombination Strg + C (Windows/Mac) oder schließen Sie einfach das Terminal-Fenster.
